    [GET] https://www.tornstats.com/api/v1/{key}/settings&comparespy=[1/0]&comparepersonal=[1/0]&personalstats=xantaken,attackswon,refills,statenhancersused
    Updates the player's preferences for comparing details on profiles.
    spy decides whether to display spy details or not.
    comparespy decides whether to display the stat differences in spies.
    comparepersonal decides whether to display personal stats or not.
    personalstats is a comma separated string of API-friendly references to personal stats. The order in which they're entered in the string is the order in which they'll appear on profiles. 
            {
    "status": true,
    "message": "Your settings have been updated!"
}

    [POST] https://www.tornstats.com/api/v1/{key}/store/spy
    Post must include the following parameters: 'player_id', 'player_name', 'player_level' The other optional parameters are: 'strength', 'defense', 'speed', 'dexterity', 'total', 'player_faction' To enter faction spies, the user must have the faction permission (API Save Spies) or be a Faction Assistant or above.

    Change Log:
    • Changed the spy end point. The url changed from /{KEY}/spy/{USER_ID} to /{KEY}/spy/user/{USER_ID}
    • Added support to get spies for a full roster at /{KEY}/spy/faction/{FACTION_ID}
